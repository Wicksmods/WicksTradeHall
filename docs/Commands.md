# Slash Commands

| Command | Effect |
|---|---|
| `/wth` | Toggle the Trade Hall |
| `/wth show` | Open the Trade Hall |
| `/wth hide` | Close the Trade Hall |
| `/wth options` | Open options panel |
| `/wth clear` | Clear the current listings cache |

## Minimap button

- **Left-click** — toggle.
- **Right-click** — options.
- **Drag** — reposition on the minimap ring.

## Row actions

- **Left-click** — whisper the lister.
- **Hover** — item-link tooltips.
- **Right-click** — copy message / report as miscategorized.
