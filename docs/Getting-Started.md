# Getting Started

## Install

- **CurseForge client** — search for "Wick's Trade Hall".
- **Manual** — download the latest ZIP from [CurseForge](https://www.curseforge.com/wow/addons/trade-hall) or [GitHub Releases](https://github.com/jspliff/WicksTradeHall/releases), extract the `WicksTradeHall` folder into `World of Warcraft\_classic_\Interface\AddOns\`.

## First run

1. Log in to a character in a major city (you need the `/2` Trade channel joined).
2. `/wth` — opens the Trade Hall.
3. Messages from Trade chat start populating the board, classified into categories.

## Reading a row

Each listing shows:

- **Category badge** (colored) — WTS, WTB, WTT, ENCHANT, CRAFT, TRAVEL, MISC
- **Character name** — class-colored
- **Message body** — the raw text, with any item links clickable for tooltips
- **Age timer** — 31s, 5m, 1h — listings fade white → yellow → grey as they get stale

## Filtering

Click any category tab at the top to filter to just that category. Click **ALL** to see everything.

## Minimap button

- **Left-click** — toggle the Trade Hall.
- **Right-click** — open options.
- **Drag** — move the minimap button around the ring.
