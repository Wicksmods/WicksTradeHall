# FAQ

### Why aren't I seeing any listings?

You need to be in a city where the `/2` Trade channel is joined (Shattrath, Orgrimmar, Stormwind, Ironforge, Undercity, Thunder Bluff, Darnassus, Exodar, Silvermoon). If you're in the field, you won't see Trade — that's a WoW limitation, not the addon.

### How long do listings stay?

Configurable — default is 10 minutes. Set in options.

### A listing is in the wrong category

Open an issue with the raw text. Category keywords live in `Categories.lua` and are easy to adjust.

### Does this replace Trade chat entirely?

It doesn't suppress the chat channel — it reads alongside. Some players hide the Trade channel in their chat windows after installing so they only see the bulletin board. That's your call.

### Can I see my own listings?

Yes — your own Trade messages appear in the board too, tagged with your class color.

### Does it work in Dalaran / Shattrath / capital cities?

Any city where you have `/2` joined. Shattrath in TBC is the most active.

### Is this for Era / Season of Discovery / Retail?

TBC Classic only (Interface 20505).
